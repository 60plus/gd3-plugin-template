# Contributing to GD3 Plugin Template

## Reporting Bugs

1. **Search first** - check [existing issues](https://github.com/60plus/gd3-plugin-template/issues)
2. **Use the Bug Report template** - fill in all required fields
3. **Include version info** - GamesDownloader version

## Requesting Features

1. **Search first** - someone may have already suggested it
2. **Use the Feature Request template**
3. **One feature per issue**

## Getting Help

Use the **Plugin Development Help** template if you need assistance building a plugin.

Resources:
- [README](README.md) - Full plugin development guide
- [Hook Reference](docs/HOOKS.md) - All hook specs, window.__GD__ API, CSS variables
- [Wiki](https://github.com/60plus/gd3-plugin-template/wiki) - Detailed guides per plugin type
- [Plugin Store](https://github.com/60plus/gd3-plugin-store) - Browse existing plugins

## Contributing Code

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test with GamesDownloader
5. Submit a pull request

Please follow existing code style and include comments where helpful.
